КАК ОБНОВИТЬ ВИДЕО-ОТЗЫВЫ (через YouTube)
=========================================

Видео-отзывы подтягиваются с YouTube по ссылкам — локальные файлы не нужны.
Обложка ролика берётся с YouTube автоматически, по клику открывается плеер на сайте.

ГДЕ МЕНЯТЬ ССЫЛКИ
-----------------
Откройте index.html и найдите в начале тега <script> список YT_REVIEWS:

    var YT_REVIEWS = [
      { url: "", name: "Отзыв ученицы", note: "Групповое обучение" },
      { url: "", name: "Отзыв ученика",  note: "Индивидуальное обучение" },
      { url: "", name: "Отзыв ученицы", note: "Наставничество" }
    ];

В поле  url  вставьте ссылку на ролик. Подойдёт ЛЮБОЙ формат:
    https://youtu.be/XXXXXXXXXXX
    https://www.youtube.com/watch?v=XXXXXXXXXXX
    https://www.youtube.com/shorts/XXXXXXXXXXX
    (или просто ID ролика — 11 символов)

    name  — подпись (имя / кто оставил отзыв)
    note  — формат обучения или короткое описание

Пример заполненной строки:
    { url: "https://youtu.be/dQw4w9WgXcQ", name: "Анна", note: "Групповое обучение" },

ДОБАВИТЬ БОЛЬШЕ ОТЗЫВОВ
-----------------------
Скопируйте любую строку { url: "...", name: "...", note: "..." } и вставьте ещё —
карточек может быть сколько угодно, они сами выстроятся в сетку.

УБРАТЬ КАРТОЧКУ
---------------
Удалите её строку из списка. Либо оставьте url пустым ("") — тогда карточка
покажет бейдж «Скоро» (заглушка).

ПОСЛЕ ИЗМЕНЕНИЙ
---------------
Сохраните index.html -> сделайте commit и push -> сайт обновится за 1-2 минуты.
Или просто пришлите ссылки на ролики — вставлю и опубликую сам.

ПОДСКАЗКА ПРО YOUTUBE
---------------------
Ролик на YouTube может быть «Не для всех» (Unlisted) — тогда его не видно в поиске,
но он отлично встраивается на сайт по ссылке. Не ставьте «Доступ ограничен» (Private) —
такой ролик встроить нельзя.
