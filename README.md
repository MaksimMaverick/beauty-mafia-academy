# Beauty Mafia Hair Academy

Одностраничный сайт академии обучения лечению и модификации волос (кератин, ботокс, реконструкция, нанопластика) — Chicago.

## Как обновить сайт
1. Отредактируйте `index.html` (или замените картинки в `images/`).
2. Запушьте изменения — GitHub Pages обновится автоматически за 1–2 минуты.

## Видео-отзывы (YouTube)
Ссылки на ролики задаются в списке `YT_REVIEWS` в начале тега `<script>` в `index.html`.
Вставьте ссылку YouTube (`youtu.be/…`, `watch?v=…`, `shorts/…` или ID) в поле `url`.
Пустой `url` = карточка «Скоро». Подробно — в `Как-обновить-видео-отзывы.txt`.

## Форма заявок
Заявки уходят в Telegram-бот. Настройки — в начале тега `<script>` в `index.html`
(`TG_BOT_TOKEN`, `TG_CHAT_ID`). Если поля пустые — заявки уходят на e-mail.
